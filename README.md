Elixir LINE Bot
===

The LINE Messaging API implement for Elixir

## Installation

Adding `linebot` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:linebot, "~> 0.1.0"}
  ]
end
```
